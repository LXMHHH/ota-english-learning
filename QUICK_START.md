# 🚀 快速部署清单

## ✅ 5分钟部署到Streamlit Cloud

### 步骤1：上传到GitHub（2分钟）
1. 登录 https://github.com
2. 创建新仓库 `ota-english-learning`（设为Public）
3. 上传这个文件夹里的所有文件：
   - `app.py`
   - `requirements.txt`
   - `.streamlit/config.toml`

### 步骤2：部署到Streamlit（2分钟）
1. 访问 https://share.streamlit.io/
2. 用GitHub账号登录
3. 点击 "New app"
4. 选择你的仓库和 `app.py` 文件
5. 点击 "Deploy!"

### 步骤3：添加到手机（1分钟）
**iPhone:**
Safari → 分享 → 添加到主屏幕

**Android:**
Chrome → 菜单 → 添加到主屏幕
